The following Automated tests are written in JavaScript, using Selenium through the Jest framework.
Please note that the node_modules folder will not be present in the final version of the archive.

To download all dependencies, please run the following command in the terminal before running any tests:
npm install

The Design Pattern used for these automated tests is Page Object Model (POM).
To run tests, use one of the following commands in the terminal:
- npm test
- npm test [file name inside the '/tests' folder] -> Ex. npm test Search
